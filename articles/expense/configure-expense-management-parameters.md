# <a name="configure-expense-management-parameters"></a>配置支出管理参数

_**适用于：** 面向资源/非库存场景的 Project Operations_

此主题介绍控制支出管理中的常规行为的参数。

## <a name="general"></a>常规

| 字段                                                    | 描述 |
|----------------------------------------------------------|-------------|
| 标准里程比率                                 | 输入里程支出的报销比率。 此比率乘以为支出输入的里程计算要报销的出差支出金额。 |
| 验证支出用途                                 | 启用此选项可将用户限制为用户在创建支出报表时在 **支出用途报表** 字段中配置的一组现有值。 |
| 个人支出支付人                                | 选择谁负责支付归类为个人支出的信用卡交易金额。 |
| 在追溯上显示整个支出报表               | 选择此选项可在查看过帐支出报表时生成的特定凭证的原始单据详细信息时显示支出报表的所有支出。 |
| 出差必须预授权                 | 选择此选项将需要先提交并批准出差申请，然后员工才可以提交支出报表。 |
| 允许在过帐前编辑分配            | 选择支出报表的分配在过帐之前是否可以编辑。 |
| 评估支出管理策略                     | 选择何时评估支出以确定是否违反了支出策略。 输入并保存支出条目或将支出提交以进行审批时，可以评估支出是否违规。 保存支出行时，将评估所需收据的策略规则。 |
| 允许内部公司支出行                         | 选择是否可以在支出报表中输入其他法人的支出。 |
| 允许编辑信用卡支出的汇率 | 选择此选项可以允许用户编辑导入的信用卡支出的汇率。 |
| 要显示的多级层次结构字段                  | 选择当工作流分配类型设置为层次结构并且 **层次结构** 选择设置为使用审批层次结构支出多级审批时显示的审批者字段。 当多级审批层次结构用于工作流时，所选字段将显示在支出报表上，并且可以编辑。 |
| 输入员工信用卡号                        | 选择是否可以输入 15 个字符或 16 个字符的卡号，并将其保存在员工 **信用卡** 页的 **卡 ID** 字段中。 |
| 验证出差申请目的                      | 启用此选项可将用户限制为用户在创建出差申请时在 **支出用途报表** 字段中配置的一组现有值。 |

## <a name="financial"></a>金融业

| 字段                                                                                    | 描述 |
|------------------------------------------------------------------------------------------|-------------|
| 分类帐日记帐名称                                                                | 选择要向其过帐批准的支出报表的分类帐日记帐的名称。 |
| 启用从支出退税                                                        | 选择此选项可为符合条件的支出启用支出退税。 如果启用了美国销售税和使用税规则，则不能选择此选项。 |
| 立即过帐预付现金                                                           | 选择此选项可以在完成支付和转帐流程后过帐批准的预付现金。 如果未选择此选项，支付和转帐流程流程将生成未过帐的一般日记帐。 |
| 更正过帐期间的会计日期                                                   | 如果当前期间为暂停或关闭，选择此选项可在过帐支出时更新会计日期。 会计日期将被设置为下一个未结期间。 |
| 允许根据付款方式中指定的抵销科目对交易进行分组       | 选择此选项可基于支出的付款方式中指定的抵销科目汇总支出报表的支出交易。 |
| 含税                                                                             | 选择此选项默认将销售税包括在支出金额中。 |
| 关闭出差申请时释放担保款                                      | 选择此选项可在关闭批准的出差申请时释放担保资金。 |
| 允许在预算登记和项目预算的预算超支时提交出差申请 | 选择此选项可允许员工提交出差申请以供审批，即使他们超出预算登记中设置的允许预算或项目允许的预算。 |
| 允许在预算登记和项目预算的预算超支时提交支出报表     | 选择此选项可允许员工提交支出报表以供审批，即使他们超出预算登记中设置的允许预算或项目允许的预算。 |
| 仅允许在预算登记的预算超支时审批支出报表                 | 选择此选项可允许审批者审批超出预算登记中设置的允许预算的支出报表。 |

## <a name="per-diem"></a>每日

| 字段                                 | 描述 |
|---------------------------------------|-------------|
| 出差津贴的最小工时数            | 输入若要有资格获得出差相关支出的出差津贴，员工每天必须工作的默认最小工时数。 此值仅用作出差津贴费率层的 **最小工时数** 字段的默认值。 |
| 餐费百分比                          | 当 **计算餐费扣减** 字段设置为 **每日用餐类型** 或 **每天用餐次数** 时，输入出差相关支出第一天和最后一天在餐饮上使用的出差津贴的默认百分比。 第一天和最后一天的工作日可能比标准工作日短。 因此，这些日期的出差津贴金额可能与标准金额不同。 如果将百分比设置为 **0**（零），第一天和最后一天的扣减将为 0.00。 |
| 住宿百分比                         | 输入在旅行相关支出的第一天和最后一天在住宿上使用的出差津贴的默认百分比。 第一天和最后一天的工作日可能比标准工作日短。 因此，这些日期的出差津贴金额可能与标准金额不同。 如果将百分比设置为 **0**（零），第一天和最后一天的扣减将为 0.00。 |
| 其他百分比                         | 输入在旅行相关支出的第一天和最后一天在杂项支出中使用的出差津贴的默认百分比。 第一天和最后一天的工作日可能比标准工作日短。 因此，这些日期的出差津贴金额可能与标准金额不同。 如果将百分比设置为 **0**（零），第一天和最后一天的扣减将为 0.00。 |
| 早餐减少百分比 | 输入在早餐上减少的出差津贴的金额。 例如，如果一名员工享受免费早餐，您可能希望将出差津贴的金额减少 10%。 |
| 午餐减少百分比     | 输入在午餐上减少的出差津贴的金额。 例如，如果一名员工享受免费午餐，您可能希望将出差津贴的金额减少 15%。 |
| 晚餐减少百分比    | 输入在晚餐上减少的出差津贴的金额。 例如，如果一名员工享受免费晚餐，您可能希望将出差津贴的金额减少 25%。 |
| 计算餐费扣减           | 通过选择扣减是基于每次旅行或每天的用餐类型，还是基于每天允许的用餐次数，来指定系统应如何计算出差津贴扣减。 |
| 出差津贴舍入                     | 选择用于出差津贴支出的舍入类型。 如果选择普通舍入，任何具有 0.50 金额的出差津贴支出都将向上舍入到 1.00，任何具有 0.50 以下金额的出差津贴支出都将向下舍入到 0.00。 |
| 基于出差津贴计算          | 选择出差津贴金额是基于日历日还是基于 24 小时周期。 |

## <a name="fax-cover-pages"></a>传真封面

| 字段                          | 描述 |
|--------------------------------|-------------|
| 指令                   | 输入员工为用于发送与支出报表相关的收据的传真创建封面时必须遵循的说明。 要输入将根据用户语言显示的语言特定文本，请选择 **翻译**。 |
| 用户 ID（条码信息） | 选择此选项可将员工的唯一标识符存储在传真封面上使用的条码中。 |
| 条码                       | 选择在传真的封面上使用的条码。 支出管理中支持条码 39 和 128。 |

## <a name="anti-corruption"></a>反腐败

| 字段                                 | 描述 |
|---------------------------------------|-------------|
| 显示反腐败证明   | 选择此选项可在创建支出报表时显示反腐败文本。 然后可以启用将需要在支出报表上选择反腐败证明的特定支出类别。 例如，与政府官员支出有关的礼品类别可能要求员工确认支出符合与政府官员有关的公司政策。 |
| 向提交者显示的反腐败消息 | 输入要为创建支出报表的员工显示的文本。 要输入将根据用户语言显示的语言特定文本，请选择 **翻译**。 |
| 向审批者显示的反腐败消息  | 输入创建支出报表时应显示给审批者的文本。 要输入将根据用户语言显示的语言特定文本，请选择 **翻译**。 |
