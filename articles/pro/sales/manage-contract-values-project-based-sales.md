<span data-ttu-id="00cdd-182">规则 1：如果 **包含的任务** 字段为空或设置为 **所有项目任务**，项目的所有任务都将包含在合同子项中。</span><span class="sxs-lookup"><span data-stu-id="00cdd-182">Rule 1: If the **Included Tasks** field is blank or set to **All Project Tasks**, all tasks of the project are included on the contract line.</span></span>
