# <a name="configure-chargeable-components-of-a-project-based-contract-line---lite"></a><span data-ttu-id="f0a71-103">配置基于项目的合同子项的应计费组件 - 精简</span><span class="sxs-lookup"><span data-stu-id="f0a71-103">Configure chargeable components of a project-based contract line - lite</span></span>

<span data-ttu-id="f0a71-104">_**适用于：** 精简部署 - 估价交易开票_</span><span class="sxs-lookup"><span data-stu-id="f0a71-104">_**Applies To:** Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="f0a71-105">基于项目的合同子项具有 *包含* 组件和 *应计费* 组件。</span><span class="sxs-lookup"><span data-stu-id="f0a71-105">A project-based contract line has *included* components and *chargeable* components.</span></span>

<span data-ttu-id="f0a71-106">包含组件是受以下各项约束的组件：</span><span class="sxs-lookup"><span data-stu-id="f0a71-106">Included components are components that are subject to:</span></span>

  - <span data-ttu-id="f0a71-107">计费方法和客户拆分</span><span class="sxs-lookup"><span data-stu-id="f0a71-107">Billing method and customer splits</span></span>
  - <span data-ttu-id="f0a71-108">上限</span><span class="sxs-lookup"><span data-stu-id="f0a71-108">Not-to-exceed limits</span></span> 
  - <span data-ttu-id="f0a71-109">基于项目的合同子项上定义的发票频率设置</span><span class="sxs-lookup"><span data-stu-id="f0a71-109">Invoice frequency settings defined on the project-based contract line</span></span>

<span data-ttu-id="f0a71-110">包含组件的子集可以使用 **记帐类型** 字段标记为应计费。</span><span class="sxs-lookup"><span data-stu-id="f0a71-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="f0a71-111">**记帐类型** 字段是一个选项集，允许在合同子项的上下文中使用以下值：</span><span class="sxs-lookup"><span data-stu-id="f0a71-111">The **Billing Type** field is an option-set that allows the following values in the context of a contract line:</span></span>

  - <span data-ttu-id="f0a71-112">应计费</span><span class="sxs-lookup"><span data-stu-id="f0a71-112">Chargeable</span></span>
  - <span data-ttu-id="f0a71-113">非应计费</span><span class="sxs-lookup"><span data-stu-id="f0a71-113">Non-chargeable</span></span>

<span data-ttu-id="f0a71-114">应计费组件可以在任务、角色和交易类别中定义。</span><span class="sxs-lookup"><span data-stu-id="f0a71-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="f0a71-115">应计费在项目合同子项的任务中定义，应用于合同子项中包括的所有交易类。</span><span class="sxs-lookup"><span data-stu-id="f0a71-115">Chargeability is defined on tasks for a project contract line and applies to all transaction classes included on the line.</span></span> <span data-ttu-id="f0a71-116">如果合同子项上的 **包含任务** 字段为空或设置为 **整个项目**，**应计费任务** 选项卡将不可用。</span><span class="sxs-lookup"><span data-stu-id="f0a71-116">If the **Include Tasks** field on a contract line is blank or set to **Entire project**, the **Chargeable tasks** tab will not be available.</span></span>

<span data-ttu-id="f0a71-117">在项目合同子项的角色中定义的应计费仅应用于 **时间** 交易类。</span><span class="sxs-lookup"><span data-stu-id="f0a71-117">Chargeability defined on roles for a project contract line only applies to the **Time** transaction class.</span></span> <span data-ttu-id="f0a71-118">如果合同子项上的 **包含时间** 字段设置为 **否**，**应计费角色** 选项卡将不可用。</span><span class="sxs-lookup"><span data-stu-id="f0a71-118">If the **Include Time** field on a contract line is set to **No**, the **Chargeable roles** tab will not be available.</span></span>

<span data-ttu-id="f0a71-119">在项目合同子项的交易类别中定义的应计费仅应用于 **支出** 交易类。</span><span class="sxs-lookup"><span data-stu-id="f0a71-119">Chargeability defined on transaction categories for a project contract line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="f0a71-120">如果 **包含支出** 字段设置为 **否**，**应计费类别** 选项卡将不可用。</span><span class="sxs-lookup"><span data-stu-id="f0a71-120">If the **Include Expenses** field is set to **No**, the **Chargeable Categories** tab will not be available.</span></span>

### <a name="update-a-project-task-as-chargeable-or-non-chargeable"></a><span data-ttu-id="f0a71-121">将项目任务更新为应计费或非应计费</span><span class="sxs-lookup"><span data-stu-id="f0a71-121">Update a project task as chargeable or non-chargeable</span></span>

<span data-ttu-id="f0a71-122">项目任务在特定合同子项上可以是应计费或非应计费，这样可以进行以下设置：</span><span class="sxs-lookup"><span data-stu-id="f0a71-122">A project task can be chargeable or non-chargeable on a specific contract line which makes the following setup possible:</span></span>

<span data-ttu-id="f0a71-123">如果基于项目的合同子项包含 **时间** 和某个任务，**T1** 将作为应计费与其关联。</span><span class="sxs-lookup"><span data-stu-id="f0a71-123">If a project-based contract line includes **Time** and a certain task, **T1** is associated to it as chargeable.</span></span> <span data-ttu-id="f0a71-124">如果有另一个包含 **支出** 的合同子项，您可以将该合同子项上的 T1 任务作为非应计费关联。</span><span class="sxs-lookup"><span data-stu-id="f0a71-124">If there is a second contract line that includes **Expense**, you can associate the T1 task on the contract line as non-chargeable.</span></span> <span data-ttu-id="f0a71-125">结果是，任务中记录的所有时间都为应计费，所有支出都为非应计费。</span><span class="sxs-lookup"><span data-stu-id="f0a71-125">The result is that all of the time recorded on the task is chargeable and all expenses are non-chargeable.</span></span>
