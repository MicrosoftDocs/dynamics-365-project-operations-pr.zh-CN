# <a name="configure-the-chargeable-components-of-a-project-based-quote-line"></a><span data-ttu-id="b8cf2-103">配置基于项目的报价单明细的应计费组件</span><span class="sxs-lookup"><span data-stu-id="b8cf2-103">Configure the chargeable components of a project-based quote line</span></span>

<span data-ttu-id="b8cf2-104">_**适用于：** 面向资源/非库存场景的 Project Operations_</span><span class="sxs-lookup"><span data-stu-id="b8cf2-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios_</span></span>

<span data-ttu-id="b8cf2-105">基于项目的报价单明细可能具有所含组件和应计费组件。</span><span class="sxs-lookup"><span data-stu-id="b8cf2-105">A project-based quote line can have included components and chargeable components.</span></span>

<span data-ttu-id="b8cf2-106">所含组件受基于项目的报价单明细上定义的计费方法、客户拆分、上限以及发票频率设置的约束。</span><span class="sxs-lookup"><span data-stu-id="b8cf2-106">Included components are subject to the billing method, customer splits, not-to-exceed limits, and invoice frequency settings defined on the project-based quote line.</span></span>
<span data-ttu-id="b8cf2-107">可以使用 **记帐类型** 将所含组件的子集标记为应计费。</span><span class="sxs-lookup"><span data-stu-id="b8cf2-107">A subset of the included components can be marked as chargeable by using **Billing Type**.</span></span> <span data-ttu-id="b8cf2-108">您可以在报价单明细上下文内的 **记帐类型** 字段中选择以下选项之一：</span><span class="sxs-lookup"><span data-stu-id="b8cf2-108">You can select one of the following options in the **Billing Type** field in the context of a quote line:</span></span>

   - <span data-ttu-id="b8cf2-109">应计费</span><span class="sxs-lookup"><span data-stu-id="b8cf2-109">Chargeable</span></span>
   - <span data-ttu-id="b8cf2-110">非应计费</span><span class="sxs-lookup"><span data-stu-id="b8cf2-110">Non-chargeable</span></span>

<span data-ttu-id="b8cf2-111">应计费组件可以在角色和交易类别中定义。</span><span class="sxs-lookup"><span data-stu-id="b8cf2-111">Chargeable components can be defined on roles and transaction categories.</span></span>

<span data-ttu-id="b8cf2-112">针对项目报价单明细的角色定义的应计费仅适用于 **时间** 交易类。</span><span class="sxs-lookup"><span data-stu-id="b8cf2-112">Chargeability that is defined on roles for a project quote line only applies to the **Time** transaction class.</span></span> <span data-ttu-id="b8cf2-113">如果项目报价单明细的 **包括时间** = **无**，则 **应计费角色** 选项卡不可用。</span><span class="sxs-lookup"><span data-stu-id="b8cf2-113">If a project quote line has **Include Time** = **No**, the **Chargeable Roles** tab isn't available.</span></span>

<span data-ttu-id="b8cf2-114">针对项目报价单明细的交易类别定义的应计费仅适用于 **支出** 交易类。</span><span class="sxs-lookup"><span data-stu-id="b8cf2-114">Chargeability that is defined on transaction categories for a project quote line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="b8cf2-115">如果项目报价单明细的 **包括支出** = **无**，则 **应计费类别** 选项卡不可用。</span><span class="sxs-lookup"><span data-stu-id="b8cf2-115">If a project quote line has **Include Expenses** = **No**, the **Chargeable Categories** tab isn't available.</span></span>

## <a name="update-a-role-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="b8cf2-116">将角色更新为应计费或非应计费</span><span class="sxs-lookup"><span data-stu-id="b8cf2-116">Update a role to be chargeable or non-chargeable</span></span>

## <a name="update-a-transaction-category-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="b8cf2-120">将交易类别更新为应计费或非应计费</span><span class="sxs-lookup"><span data-stu-id="b8cf2-120">Update a transaction category to be chargeable or non-chargeable</span></span>
