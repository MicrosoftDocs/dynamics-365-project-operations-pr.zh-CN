# <a name="configure-accounting-for-billable-projects"></a><span data-ttu-id="4f5a4-103">配置收费项目会计</span><span class="sxs-lookup"><span data-stu-id="4f5a4-103">Configure accounting for billable projects</span></span>

<span data-ttu-id="4f5a4-104">_**适用于：** 基于资源/非库存场景的 Project Operations，精简部署 - 估价开票交易_</span><span class="sxs-lookup"><span data-stu-id="4f5a4-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="4f5a4-105">Dynamics 365 Project Operations 支持收费项目（包括时间和材料以及固定价格交易）的各种会计选项。</span><span class="sxs-lookup"><span data-stu-id="4f5a4-105">Dynamics 365 Project Operations supports various accounting options for billable projects that include time and material and fixed price transactions.</span></span>

- <span data-ttu-id="4f5a4-106">**时间和材料交易**：随着工作的进行，这些交易根据项目的时间、支出、项目或费用的消耗开票。</span><span class="sxs-lookup"><span data-stu-id="4f5a4-106">**Time and material transactions**: These transactions are invoiced as the work progresses based on the consumption of hours, expenses, items, or fees on the project.</span></span> <span data-ttu-id="4f5a4-107">这些交易成本可以与每笔交易的收入相匹配，项目随着工作的进展开票。</span><span class="sxs-lookup"><span data-stu-id="4f5a4-107">These transaction costs can be matched with the revenue on each transaction and the project is invoiced as work progresses.</span></span> <span data-ttu-id="4f5a4-108">项目收入也可以在交易发生时计入应计。</span><span class="sxs-lookup"><span data-stu-id="4f5a4-108">Project revenue can be also accrued at the time when the transaction occurs.</span></span> <span data-ttu-id="4f5a4-109">在开票过程中，会确认收入并冲销应计收入（如果适用）。</span><span class="sxs-lookup"><span data-stu-id="4f5a4-109">During invoicing, revenue is recognized and if applicable, accrued revenue is reversed.</span></span>

<span data-ttu-id="4f5a4-112">如果一个项目与一个或多个合同子项关联，该项目被视为可计费。</span><span class="sxs-lookup"><span data-stu-id="4f5a4-112">A project is considered billable when it is associated with one or more contract lines.</span></span> <span data-ttu-id="4f5a4-113">项目合同子项为其自身定义允许的计费方法和交易类型。</span><span class="sxs-lookup"><span data-stu-id="4f5a4-113">A project contract line defines for itself which billing method and transaction types are allowed.</span></span>
