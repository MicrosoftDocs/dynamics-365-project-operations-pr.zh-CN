# <a name="view-an-expense-report-from-general-ledger-or-procurement-and-sourcing"></a>从总帐或采购查看支出报表

在总帐或采购中运行查询以查看交易记录时，查询可能会返回过多信息。 因此，可能需要缩小搜索范围。 若要获取所需的信息，您可以深化到交易记录以查看它将出现的支出报表。

对于支出报表，您可以从交易记录向下钻取到存储在支出管理中的原始报表。 除了支出总额，还可以查看逐项交易、适用于增值税 (VAT) 回收的交易记录，以及附加到支出的收货。

若要向下钻取到交易记录所在的原始支出报表，请选择交易记录，然后选择 **原始文档**。

在查看各个交易记录时，您还可以查看任何附加到交易记录的附加数据。 例如，对于列出的酒店支出明细，您可以查看支出的详细列表，如 Internet 访问权限、电话和房间服务。
